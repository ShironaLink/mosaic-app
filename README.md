# MosaicApp - 画像モザイクアプリ

画像にモザイクをかけたり、スポイトで色を取得してペイントできるシンプルなデスクトップアプリです。

## 機能

- **ブラシモザイク** — ドラッグした部分にモザイクを適用
- **範囲選択モザイク** — 矩形で囲んだ範囲にモザイクを適用
- **スポイト** — 画像から色を取得
- **ペイント** — 取得した色で自由に描画
- **カラースウォッチ** — クリックでカラーチューザーから任意の色を選択
- **ズーム / パン** — マウスホイールでズーム、右クリックドラッグでパン
- **ドラッグ＆ドロップ** — 画像ファイルをウィンドウに直接ドロップ
- **Undo** — Ctrl+Z で最大30回まで元に戻せる

## インストール

### exe版（Windows）

[Releases](../../releases) から最新版をダウンロードして、`MosaicApp.exe` を実行してください。Python のインストールは不要です。

### Python から実行

```bash
pip install Pillow windnd
python mosaic_app.py
```

## 操作方法

| 操作 | 説明 |
|------|------|
| 左クリック / ドラッグ | 選択中のツールを使用 |
| 右クリックドラッグ | パン（画像の移動） |
| マウスホイール | ズーム |
| Ctrl+O | 画像を開く |
| Ctrl+S | 保存 |
| Ctrl+Z | 元に戻す |
| Ctrl+0 | 画面にフィット |
