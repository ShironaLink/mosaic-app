# MosaicApp - 画像モザイクアプリ

画像にモザイクをかけたり、トリミングしたりできるシンプルなデスクトップアプリです。

## 機能

- **ブラシモザイク** — ドラッグした部分にモザイクを適用
- **範囲選択モザイク** — 矩形で囲んだ範囲にモザイクを適用
- **四角トリミング** — 矩形で切り抜き（Ctrl押しで正方形に制約）
- **円形トリミング** — 楕円で切り抜き（Ctrl押しで正円に制約）
- **トリミング範囲の移動** — 選択範囲をドラッグで移動可能
- **ズーム / パン** — マウスホイールでズーム、右クリックドラッグでパン
- **ドラッグ＆ドロップ** — 画像ファイルをウィンドウに直接ドロップ
- **多フォーマット保存** — PNG / JPEG / GIF / BMP / TIFF / WebP に対応
- **保存サイズ調整** — 10%〜200% でリサイズして保存可能
- **推定容量表示** — 保存前にファイルサイズの目安を確認
- **Undo** — Ctrl+Z / ⌘+Z で最大30回まで元に戻せる

## インストール

### macOS（おすすめ）

1. [Releases](../../releases) ページから **`MosaicApp-macOS.zip`** をダウンロード
2. ZIPを展開すると `モザイクアプリ.app` が出てきます
3. `モザイクアプリ.app` を **アプリケーション** フォルダにドラッグ
4. Launchpad またはアプリケーションフォルダからダブルクリックで起動

> **初回起動時の注意**: 「開発元が未確認」と表示された場合は、`モザイクアプリ.app` を右クリック →「開く」を選択してください。

**必要なもの**: Python 3 と Pillow が必要です（macOS にはPython 3がプリインストールされています）。Pillowが未インストールの場合は、ターミナルで以下を実行してください:

```bash
pip3 install Pillow
```

### Windows（exe版）

1. [Releases](../../releases) ページから **`MosaicApp.exe`** をダウンロード
2. ダウンロードした `MosaicApp.exe` をダブルクリックで起動

> **初回起動時の注意**: Windows Defender SmartScreen が警告を表示する場合は、「詳細情報」→「実行」をクリックしてください。

Python のインストールは不要です。

### Python から実行（開発者向け）

```bash
# macOS
pip3 install Pillow pyobjc
python3 mosaic_app_macos.py

# Windows
pip install Pillow windnd
python mosaic_app.py
```

## 操作方法

| 操作 | 説明 |
|------|------|
| 左クリック / ドラッグ | 選択中のツールを使用 |
| 右クリックドラッグ | パン（画像の移動） |
| マウスホイール | ズーム |
| Ctrl / ⌘ + ドラッグ | トリミング時に正方形/正円に制約 |
| Ctrl+O / ⌘+O | 画像を開く |
| Ctrl+S / ⌘+S | 保存 |
| Ctrl+Z / ⌘+Z | 元に戻す |
